# Editeur Tableau et Assembleur 
Un éditeur de tableau assembleur de codes Latex développé en HTML , CSS et Javascript

##Fonctionnalités 
### Editeur Tableau 
- fournir un tableau  manipulable et à remplir .
- Génere automatiquement du code latex dans l'environement "array".
- permettant de copier le code generé .

### Assembleur
- offrir un environement de code en latex
- combine les codes dans deux environements séparés (array et aligned)
- ajoute automatiquement le séparateur "&" à chaque nouvelle ligne et "\\" à la fin.

## Structure du projet                                                                                                                 
- index.html 
- style.css
- script.js

## Utilisation
Ouvrez le fichier 'index.html' dans votre navigateur web.
